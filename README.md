Hospital Management System
A simple yet powerful Hospital Management System built with Python and MySQL, designed to demonstrate core concepts of relational databases, CRUD operations, and modular programming. This project manages hospital workflows for doctors, patients, and non-medical staff, and includes a secure appointment booking system.
Features
Entity Management:
Separate MySQL tables for Doctors, Patients, and Non-medical Staff
CRUD operations for each entity

Appointment System:
Book, view, and update appointments
Status tracking: Pending, Approved, Cancelled

Authentication:
Login system for doctors and patients
Role-based access to functionalities

Robust Backend:
MySQL Connector for database interaction
Input validation and exception handling

Modular Codebase:
Organized for easy maintenance and scalability
Designed for integration into larger systems
